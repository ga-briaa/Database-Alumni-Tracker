## Alumni Tracker
> Database University Alumni Tracker is a database containing information and career paths about alumni from the hypothetical university, Database University (DU). <br>
> DU Alumni Tracker aims to have a streamlined view of all alumni at DU, to be able to contact alumni, view their career paths, and provide support when necessary.

### Features and Functionalities
- User login - Allows users to login and block access to the full database
- Change Table View - Access different tables efficiently
- Search Function - Search records using specific keywords in the current table view
- Filter Function - Filter records in specific category in the current table view
- Manage Data (Edit, Add, Delete) - Allows the admin user to edit, add, and delete records
- Table Header Sorter - Sort records based on the table header in ascending or descending order
- Pagination - Dynamically separates and shows 10 records per page of the current table
